# Student Management System (Hibernate)

## Description
Console-based Student Management System built using Hibernate ORM, MySQL, and Maven.

## Features
- Add Student
- Update Student
- Delete Student
- Fetch Student by ID

## Tech Stack
- Java
- Hibernate
- MySQL
- Maven
- Eclipse

## How to Run
1. Create database `studentdb`
2. Update MySQL credentials in `hibernate.cfg.xml`
3. Run `MainApp.java`
